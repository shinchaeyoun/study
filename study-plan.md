## 📚 학습 커리큘럼 (4개월 집중 과정)

### **1개월차: React 기본기**
- **주차별 목표**
  - 1주차: React 컴포넌트, JSX, Props/State 이해  
  - 2주차: 이벤트 처리, 조건부 렌더링, 리스트 렌더링  
  - 3주차: Hooks (useState, useEffect, useRef)  
  - 4주차: Context API, 컴포넌트 구조화, 작은 프로젝트(간단한 Todo 앱)  

---

### **2개월차: Next.js + 라우팅/데이터 패칭**
- **주차별 목표**
  - 1주차: Next.js 프로젝트 구조, Pages 라우팅  
  - 2주차: Dynamic Routing, API Routes  
  - 3주차: SSR(서버사이드 렌더링), SSG(정적 사이트 생성)  
  - 4주차: getServerSideProps, getStaticProps, 클라이언트 데이터 패칭 → 블로그 프로젝트 제작  

---

### **3개월차: TypeScript + Tailwind CSS**
- **주차별 목표**
  - 1주차: TypeScript 기본 문법, 타입 정의, 인터페이스/제네릭  
  - 2주차: React + TypeScript 통합 (Props, State, Hooks 타입 지정)  
  - 3주차: Tailwind CSS 기본 문법, Utility-first 스타일링  
  - 4주차: 반응형 디자인, 다크모드, 컴포넌트 스타일링 → 포트폴리오 사이트 제작  

---

### **4개월차: 종합 프로젝트 & 프리랜서 준비**
- **주차별 목표**
  - 1주차: Next.js + TypeScript + Tailwind으로 풀스택 프로젝트 기획  
  - 2주차: API 연동 (REST/GraphQL), 인증 기능 추가  
  - 3주차: 테스트(Jest, Cypress) 및 배포(Vercel, Netlify)  
  - 4주차: 포트폴리오 정리, GitHub/개인 블로그 공개, 프리랜서 플랫폼 프로필 작성

## 🛠️ 대시보드 MVP 기능 목록

### 1. **인증/로그인**
- 로그인 페이지 (이메일/비밀번호 입력)
- 가짜 API 또는 Mock 데이터로 인증 처리
- 로그인 성공 시 대시보드 메인 페이지로 이동

### 2. **메인 대시보드**
- 주요 지표 카드 3~4개 표시  
  - 총 사용자 수  
  - 총 주문 수  
  - 총 매출액  
  - 신규 가입자 수  

### 3. **데이터 테이블**
- **사용자 목록**: 이름, 이메일, 가입일, 상태(활성/비활성)  
- **주문 내역**: 주문 ID, 사용자, 금액, 상태(완료/취소/진행중)  
- 필터링/검색 기능 포함

### 4. **차트/그래프**
- 매출 추이 (월별 매출액 그래프)  
- 사용자 증가 추세 (월별 신규 가입자 수)  

### 5. **반응형 UI**
- Tailwind CSS로 모바일/데스크톱 대응  
- 다크모드 지원

## 📊 데이터 구조 설계 예시 (TypeScript 인터페이스)

// 사용자(User) 데이터 구조
interface User {

  name: string;
  email: string;
  joinedAt: string; // 가입일
  status: "active" | "inactive";
}

// 주문(Order) 데이터 구조
interface Order {
  id: number;
  userId: number;
  amount: number;
  status: "completed" | "pending" | "cancelled";
  createdAt: string;
}

// 대시보드 지표(Metrics)
interface DashboardMetrics {
  totalUsers: number;
  totalOrders: number;
  totalRevenue: number;
  newUsers: number;
}
